# Changelog

All notable changes to the BFIN SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-12

### Added
- Initial release of BFIN SDK
- TypeScript client with Axios
- React Query hooks support
- SWR hooks support
- JWT authentication with token refresh
- Full type safety with auto-generated types from OpenAPI
- Support for all BFIN API endpoints

### Features
- **Client**: Vanilla TypeScript/JavaScript client
- **React Query**: Pre-configured hooks for React applications
- **SWR**: Pre-configured hooks for SWR users
- **Authentication**: Built-in JWT token management
- **Type Safety**: Full TypeScript support
- **Auto-generated**: SDK is automatically generated from OpenAPI specification
