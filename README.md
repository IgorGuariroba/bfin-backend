# bfin-backend

Backend Fastify do bfin, dono do schema e das migrations Drizzle (ADR-0017 do `bfin-app`).

## Desenvolvimento

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — servidor com watch (tsx)
- `npm run build` — compila para `dist/`
- `npm start` — roda o build
- `npm test` — testes (Vitest)
- `npm run lint` — ESLint
- `npm run type-check` — checagem de tipos sem emitir
